This image represents the CI?CD pipeline in action. It builds, tests, and packages your Java project automatically.

SonarQube checks your code quality, ensuring that your project adheres to standards and avoids bugs.

Docker containerizes your application, making deployments consistent across environments.

JFrog stores your build artifacts (JARs, images) safely with versioning, allowing rollback and traceability.
